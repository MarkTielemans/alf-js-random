# alf-js-random

``` Javascript
var d1 = rand.randomDate();
logger.log(d1);
logger.log(rand.randomDate(d1));
logger.log(rand.randomNumber(10));
logger.log(rand.randomNumber(15, 18));
logger.log(rand.randomPositive(5));
logger.log(rand.randomString(10));
logger.log(rand.randomString(10, true, false));
logger.log(rand.randomString(10, false, true));
logger.log(rand.randomString(5, true, true));
``` 
